Nuestros servicios

Realizamos aplicaciones para móviles Android y iOS. Usamos tanto tecnología nativa (Android Studio, Xcode) como tecnología multiplataforma (React Native, Flutter, Ionic, Xamarin, entre otros). Además, si así lo requiere el cliente, usamos las últimas bases de datos (MongoDB, Firebase entre otros) para potenciar el funcionamiento de su aplicación.

Realizamos páginas web usando HTML y CSS, así como JavaScript. Elige entre más de cien diseños, así como el host en el cual quieres alojar tu página web.

Realizamos páginas web usando sistemas de gestión de contenidos (CMS, por sus siglas en inglés) como pueden ser WordPress, Joomla!, Drupal, PrestaShop, WooCommerce, entre otros.

Realizamos tiendas online usando Shopify. Instalamos el temas, el dominio, y los diversos plugins que potenciarán las compras en tu página web.

Realizamos aplicaciones Windows32, con las cuales podrás llevar a cabo, entre otras cosas, un seguimiento de sus ventas, un seguimiento sobre su inventario, etc, etc...

¿Cómo trabajamos?

Primero de todo, contacta con nosotros. Pídenos presupuesto sin compromiso, nosotros nos encargaremos de todo. Tras contactar con nosotros, sería recomendable realizar una llamada a través de Skype en la cual te daremos todo los detalles sobre el proyecto: fecha de finalización, plazos de entrega, precio final, etc, etc...

Te mantendremos informado sobre la situación del proyecto en todo momentos. Si surge algún problema a medida que realizamos el proyecto, contactaremos contigo para recomendarte las soluciones óptimas.

¿Cómo se realizan los pagos?

Los pagos se realizan a plazos, es decir, una vez tengamos hecho el presupuesto, contactaremos contigo para explicarte como se llevarán a cabo los pagos. Se suelen realizar pagos del 20%/25% del total. Ejemplo: si el proyecto tiene un precio de 100 EUROS, se realizarán cuatro pagos de 25 EUROS, o dos pagos de 50 EUROS (se firmará en el presupuesto).

Para realizar los pagos, usamos métodos como TransferWise o Bizum (este último en caso de que el cliente resida en España).

IMPORTANTE: Antes de comenzar a realizar el proyecto, es necesario abonar el primer plazo.
